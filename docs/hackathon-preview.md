# Hermes hackathon preview

During the Hermes hackathon, indx was used as a local media workspace that agents could operate through multiple surfaces:

- CLI/API for bounded scripted operations
- Hermes skills for library/admin workflows
- MCP tools for typed agent access
- local metadata and embedding storage for media experiments

The demo work explored how Hermes + indx can turn a local media library into a creative lab: ComfyUI batch review, latent-space visualization, A/V embedding experiments, and embedding-assisted breakbeat/found-sound remixing.

This repository does not contain the full indx app yet. It is a preview of the agent-integration direction while the open-source beta is prepared.
