# indx preview

indx is a local-first media workspace for artists, developers, designers, and creative technologists.

It helps you browse, search, compare, annotate, and organize large local media libraries without trapping your work in a proprietary database. The database is an index, not a jail.

The full app is not released yet. This repository is a preview home for early Hermes/agent-integration docs, hackathon materials, and release notes.

Open-source beta coming soon.

Website: https://indx.art  
Contact: dev@indx.art

## What is here now

- `skills/indx-library/` — preview Hermes skill for searching, inspecting, annotating, and organizing an indx media library.
- `skills/indx-admin/` — preview Hermes skill for watched directories, scans, jobs, and startup-scan settings.
- `docs/` — short notes on the hackathon integration and preview status.

## Status

This is an early preview, published for the Hermes hackathon. The skills and docs reflect current local development workflows and may need adjustment before the public indx beta.

Use this as a preview of the integration direction, not as a standalone app release.
